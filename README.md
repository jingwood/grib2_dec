# grib2_dec

A modified version of the GRIB2 data decoder which is originally provided by the Japan Meteorological Agency (JMA).

# Build

```shell
cd build/<platform-name>
make
```

# Run

```
grib2_dec <GRIB2-filename> [-xpm]
```
